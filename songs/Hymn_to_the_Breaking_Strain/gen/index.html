Hymn to the Breaking Strain

By Rudyard Kipling

Abide the twin damnation, to fail and know we fail

Some solstices have used the Leslie Fish melody. Others have used a different melody which does not seem to be recorded anywhere. Yet another option is to use Fishâ€™s melody, but combine it with darker and simpler harmonies (see 2020 version ). For several years, the Bay Solstice used some version of this abridged arrangement for three voices (note that this is in Bb minor, whereas the original is in A minor).

A recording of this from the 2020 North American Distributed Chorus is available.
